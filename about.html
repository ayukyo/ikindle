关于 iKindle

背景

iKindle 诞生于一个简单的想法：把一台闲置的 6 寸 KINDLE Paperwhite 4 改造成桌面常驻的"信息仪表盘"。墨水屏不发光、护眼、可长时间显示静态画面 —— 比任何 LCD / OLED 屏幕都更适合作为床头 / 书桌台历。

灵感来源

  • online-screensaver —— KINDLE 屏保替换项目
  • wttr.in —— 终端天气查询工具（CORS 友好）
  • MobileRead Kindle Hacking Wiki —— KWP4 浏览器兼容性资料

技术决策

项目严格遵循以下约束：

  • KINDLE 浏览器兼容：WebKit 537.36（~Chromium 28-35 时代），禁用 ES6+ / CSS 变量 / Grid / sticky
  • 零服务依赖：仅依赖 nginx + cron + bash + curl
  • 数据获取：构建期注入 + 客户端 ES5 XHR 双轨制
  • E-Ink 友好：白底 / 纯黑加粗衬线 / 远距离可读 / 整页 reload
