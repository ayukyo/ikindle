iKindle —— 6-inch Kindle E-Ink Dashboard

Transform a Kindle Paperwhite 4+ (2018 or newer) into a desktop calendar / information dashboard. Leveraging the e-ink physical properties (no backlight, eye-friendly, long-time static display, paper-like reading), perfect for bedside / desk / office as a permanent display.

✨ Features

  • 🕐 Time —— Current time + date + lunar calendar (e.g. 丙午年五月初七)
  • 📅 Calendar —— Monthly grid with today highlighted, ISO week number
  • 🌦️ Weather —— Auto-locate city by IP via wttr.in, refresh every 60s
  • 🎉 Festivals —— Chinese New Year / Dragon Boat / Mid-Autumn / National Day
  • 🌅 Sunrise/Sunset —— Real-time astronomical data
  • 💬 Hitokoto —— Daily quote (cron refreshed at 0:00)
  • 📴 Offline indicator —— Graceful degradation when network drops

🎯 Target Devices

Kindle Paperwhite 4 (2018) / KPW5 / KPW Signature / Kindle Basic 11th / Kindle Scribe (same-era Oasis / Basic 10 / Colorsoft browsers share the same WebKit 537.36 kernel and are also compatible).

📦 Tech Stack

  • Frontend: HTML5 + CSS3 + ES5 JavaScript (Kindle WebKit 537.36 compatible)
  • Build: cron + bash + sed + curl + Python cnlunar
  • Deploy: Aliyun Linux + nginx + Let's Encrypt + certbot
  • Data sources: wttr.in (IP auto-locate) + v1.hitokoto.cn + lunar lib
