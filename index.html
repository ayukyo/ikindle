iKindle —— 6 寸 KINDLE 墨水屏台历

把一台 Kindle Paperwhite 4+（2018 及更新机型）改造成桌面台历 / 信息仪表盘，利用墨水屏物理特性（不发光、护眼、可长时间显示静态画面）作为床头 / 书桌常驻屏。

✨ 功能

  • 🕐 时间 —— 当前时分 + 日期副标题 + 农历（丙午年五月初六 庚寅日）
  • 📅 日历 —— 当月月历，今日加粗高亮，ISO 周数
  • 🌦️ 天气 —— wttr.in 自动 IP 定位城市，每 60s 刷新
  • 🎉 传统节日 —— 春节 / 端午 / 中秋 / 国庆等
  • 🌅 日出日落 —— 实时天文数据
  • 💬 Hitokoto —— 每日一言（每日 0:00 cron 缓存）
  • 📴 离线提示 —— 网络断开时友好降级

🚀 快速访问

  • 📺 仪表盘 —— KINDLE 浏览器入口
  • 📖 部署指南 —— 阿里云 / 自有服务器
  • 🖼️ 截图 —— 实物展示
  • 📜 更新日志
  • ℹ️ 关于

🎯 目标设备

Kindle Paperwhite 4 (2018) / KPW5 / KPW Signature / Kindle Basic 11th / Kindle Scribe（同代 Oasis / Basic 10 / Colorsoft 浏览器内核相同，亦可兼容）。

📦 技术栈

  • 前端：HTML5 + CSS3 + ES5 JavaScript（KINDLE WebKit 537.36 兼容）
  • 构建：cron + bash + sed + curl + Python cnlunar
  • 部署：阿里云 Linux + nginx + Let's Encrypt + certbot
  • 数据源：wttr.in（IP 自动定位）+ v1.hitokoto.cn + 农历计算库
